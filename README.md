## Descripcion del pipeline

Este pipeline ETL tranforma los datos crudos de las ventas de la categoria de chocolate de la compañia 3 Patitos S.A. de forma que el departamento de ventas pueda hacer analisis de desempeño a nivel de vendedor, region, pais, y producto atravez del tiempo.
